OBJECTIF :
  Poser le socle du projet : structure du package, dependances,
  et gestion centralisee de la configuration.

FICHIERS CREES :
  - .env.example                      (modele de configuration)
  - requirements.txt                  (requests obligatoire, rich optionnel)
  - retroai_agent/__init__.py         (marque le dossier comme package Python)
  - retroai_agent/config.py           (dataclass Config + load_config())

    "Copy-Item" non reconnu. -> Solution : utiliser "copy .env.example .env".
    A retenir : verifier l'invite (C:\...> = cmd, PS C:\...> = PowerShell).

  - Chargement sans cle (validation)     -> OK (message clair + code sortie 1)
